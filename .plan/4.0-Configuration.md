# Freecode — Configuration System

## 1.0 Purpose

This document describes the configuration system for freecode, including backward compatibility with opencode configs and oh-my-openagent integration.

---

## 2.0 Configuration File Locations

### 2.1 Freecode Native (Priority Order)

| Path | Format | Purpose |
|------|--------|---------|
| `~/.config/freecode/config.yaml` | YAML | Main config |
| `~/.config/freecode/config.json` | JSON | Alternative |
| `~/.config/freecode/profiles/` | YAML | Profile configs |
| `$XDG_CONFIG_HOME/freecode/config.yaml` | YAML | XDG fallback |
| `.freecode/config.yaml` | YAML | Project-level |

### 2.2 OpenCode Compatibility (Read-Only)

| Path | Format | Purpose |
|------|--------|---------|
| `~/.config/opencode/config.json` | JSON | Main config |
| `~/.config/opencode/config.toml` | TOML | Legacy config |
| `~/.config/opencode/opencode.json` | JSON | Project config |
| `~/.config/opencode/opencode.jsonc` | JSONC | Comments |
| `~/.config/opencode/tui.json` | JSON | TUI config |

### 2.3 oh-my-openagent Integration

| Path | Format | Purpose |
|------|--------|---------|
| `~/.config/opencode/oh-my-opencode.jsonc` | JSONC | OMO main |
| `~/.config/opencode/oh-my-openagent.jsonc` | JSONC | OMO alt |

## 3.0 Config Schema

### 3.1 Core Config Fields

```go
type Config struct {
    // General
    Yolo        bool     `yaml:"yolo" json:"yolo"` // Skip confirmations
    Shell       string   `yaml:"shell" json:"shell"`
    LogLevel    string   `yaml:"logLevel" json:"logLevel"` // DEBUG|INFO|WARN|ERROR

    // Server
    Server      ServerConfig `yaml:"server" json:"server"`
    ServerPort  int          `yaml:"serverPort" json:"serverPort"`
    ServerHost  string       `yaml:"serverHost" json:"serverHost"` // Always 127.0.0.1 or ::1

    // Model
    Model       string   `yaml:"model" json:"model"`
    SmallModel  string   `yaml:"small_model" json:"small_model"`
    DefaultAgent string  `yaml:"default_agent" json:"default_agent"`

    // Features
    Snapshot    bool     `yaml:"snapshot" json:"snapshot"`
    Share       string   `yaml:"share" json:"share"` // manual|auto|disabled
    AutoUpdate  bool     `yaml:"autoupdate" json:"autoupdate"`

    // Skills
    Skills      SkillsConfig `yaml:"skills" json:"skills"`
    SkillsPaths []string     `yaml:"skillsPaths" json:"skillsPaths"`

    // Tools
    Tools       map[string]bool `yaml:"tools" json:"tools"`

    // Permissions
    Permission  PermissionConfig `yaml:"permission" json:"permission"`

    // Agents
    Agent       AgentConfig `yaml:"agent" json:"agent"`
    Agents      map[string]AgentOverride `yaml:"agents" json:"agents"`

    // Providers
    Providers   map[string]ProviderConfig `yaml:"providers" json:"providers"`

    // MCP
    MCP         map[string]MCPConfig `yaml:"mcp" json:"mcp"`

    // Commands
    Commands    map[string]CommandConfig `yaml:"commands" json:"commands"`

    // Formatting
    Formatter   FormatterConfig `yaml:"formatter" json:"formatter"`

    // LSP
    LSP         LSPConfig `yaml:"lsp" json:"lsp"`

    // Compaction
    Compaction  CompactionConfig `yaml:"compaction" json:"compaction"`

    // Experimental
    Experimental ExperimentalConfig `yaml:"experimental" json:"experimental"`

    // Instructions
    Instructions []string `yaml:"instructions" json:"instructions"`

    // oh-my-openagent settings
    OMO         *OMOConfig `yaml:"omo" json:"omo"`

    // Legacy
    DisabledProviders []string `yaml:"disabled_providers" json:"disabled_providers"`
    EnabledProviders  []string `yaml:"enabled_providers" json:"enabled_providers"`
}
```

    var raw map[string]interface{}
    if err := toml.Unmarshal(tomlData, &raw); err != nil {

    }
    return convertMap(raw), nil

### 5.3 JSONC Parsing

JSONC is JSON with comments. Strip comments before parsing:

```go
func ParseJSONC(data []byte) (any, error) {
    // Remove // comments
    // Remove /* */ comments
    return json.Unmarshal(stripComments(data))
}
```

## 6.0 Config Merging

Config sources are merged in order (later overrides):

1. Defaults
2. OpenCode configs (read-only)
3. oh-my-openagent configs
4. Freecode configs
5. Project configs
6. Environment variables
7. CLI flags

## 7.0 YOLO Mode

YOLO mode (`yolo: true`) skips all confirmations:

```go
type YOLOConfig struct {
    SkipEditConfirmations   bool `yaml:"skipEditConfirmations" json:"skipEditConfirmations"`
    SkipBashConfirmations  bool `yaml:"skipBashConfirmations" json:"skipBashConfirmations"`
    SkipDeleteConfirmations bool `yaml:"skipDeleteConfirmations" json:"skipDeleteConfirmations"`
    SkipPermissionPrompts  bool `yaml:"skipPermissionPrompts" json:"skipPermissionPrompts"`
    SkipToolConfirmations  bool `yaml:"skipToolConfirmations" json:"skipToolConfirmations"`
}
```

**Default behavior**: YOLO is `false` (safe mode)

**Commands menu toggle**: `config set yolo true`

## 9.0 Validation

Config is validated on load:

```go
func (c *Config) Validate() error {
    if c.LogLevel != "" {
        switch c.LogLevel {
        case "DEBUG", "INFO", "WARN", "ERROR":
        default:
            return fmt.Errorf("invalid logLevel: %s", c.LogLevel)
        }
    }
    if c.Share != "" {
        switch c.Share {
        case "manual", "auto", "disabled":
        default:
            return fmt.Errorf("invalid share: %s", c.Share)
        }
    }
    return nil
}
```
