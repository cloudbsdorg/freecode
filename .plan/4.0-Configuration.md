# Freecode — Configuration System

## 1.0 Purpose

This document describes the unified configuration system for freecode. All features from opencode and the oh-my-openagent plugin are built directly into freecode as a single, cohesive product.

---

## 2.0 Configuration Philosophy

**One Product, One Config:** Unlike opencode + oh-my-openagent (which requires separate config files), freecode provides a unified configuration that encompasses all features.

**Migration Compatible:** Freecode reads opencode and oh-my-openagent config files for migration purposes, but creates its own unified config.

---

## 3.0 Configuration File Locations

### 3.1 Freecode Native (Priority Order)

| Path | Format | Purpose |
|------|--------|---------|
| `~/.config/freecode/config.yaml` | YAML | **Main unified config** |
| `~/.config/freecode/config.json` | JSON | Alternative format |
| `~/.config/freecode/profiles/` | YAML | Profile configs |
| `$XDG_CONFIG_HOME/freecode/config.yaml` | YAML | XDG fallback |
| `.freecode/config.yaml` | YAML | Project-level config |

### 3.2 Legacy (Read-Only Migration)

On first run, freecode reads these configs to migrate settings:

| Path | Format | Purpose |
|------|--------|---------|
| `~/.config/opencode/config.json` | JSON | Legacy main config |
| `~/.config/opencode/config.toml` | TOML | Legacy TOML config |
| `~/.config/opencode/opencode.json` | JSON | Legacy project config |
| `~/.config/opencode/opencode.jsonc` | JSONC | Legacy JSON with comments |
| `~/.config/opencode/oh-my-opencode.jsonc` | JSONC | oh-my-openagent settings |
| `~/.config/opencode/oh-my-openagent.jsonc` | JSONC | oh-my-openagent alt config |

**Note:** These are read to migrate settings. Freecode writes only to its own config files.

## 4.0 Unified Config Schema

### 4.1 Complete Config Structure

Freecode's config is a single unified structure combining all features:

```go
type Config struct {
    // ========== General ==========
    Yolo        bool     `yaml:"yolo" json:"yolo"` // Skip all confirmations (DEFAULT: true)
    Shell       string   `yaml:"shell" json:"shell"`
    LogLevel    string   `yaml:"logLevel" json:"logLevel"` // DEBUG|INFO|WARN|ERROR

    // ========== Server ==========
    Server      ServerConfig `yaml:"server" json:"server"`
    ServerPort  int          `yaml:"serverPort" json:"serverPort"`
    ServerHost  string       `yaml:"serverHost" json:"serverHost"` // Always 127.0.0.1 or ::1

    // ========== Model ==========
    Model       string   `yaml:"model" json:"model"`
    SmallModel  string   `yaml:"smallModel" json:"smallModel"`
    DefaultAgent string  `yaml:"defaultAgent" json:"defaultAgent"`

    // ========== Features ==========
    Snapshot    bool     `yaml:"snapshot" json:"snapshot"`
    Share       string   `yaml:"share" json:"share"` // manual|auto|disabled
    YoloMode    YOLOConfig `yaml:"yoloMode" json:"yoloMode"`

    // ========== Skills ==========
    Skills      SkillsConfig `yaml:"skills" json:"skills"`
    SkillsPaths []string     `yaml:"skillsPaths" json:"skillsPaths"`

    // ========== Tools ==========
    Tools       map[string]bool `yaml:"tools" json:"tools"`
    ToolsDisabled []string       `yaml:"toolsDisabled" json:"toolsDisabled"`

    // ========== Permissions ==========
    Permission  PermissionConfig `yaml:"permission" json:"permission"`

    // ========== Agents (Built-in + Custom) ==========
    Agent       AgentConfig `yaml:"agent" json:"agent"`
    Agents      map[string]AgentOverride `yaml:"agents" json:"agents"`

    // ========== Agent Categories ==========
    Categories  map[string]CategoryConfig `yaml:"categories" json:"categories"`

    // ========== Providers ==========
    Providers   map[string]ProviderConfig `yaml:"providers" json:"providers"`
    DisabledProviders []string `yaml:"disabledProviders" json:"disabledProviders"`
    EnabledProviders  []string `yaml:"enabledProviders" json:"enabledProviders"`

    // ========== MCP (Model Context Protocol) ==========
    MCP         map[string]MCPConfig `yaml:"mcp" json:"mcp"`
    MCPDisabled []string       `yaml:"mcpDisabled" json:"mcpDisabled"`
    MCPEnvAllowlist []string   `yaml:"mcpEnvAllowlist" json:"mcpEnvAllowlist"`

    // ========== Commands ==========
    Commands    map[string]CommandConfig `yaml:"commands" json:"commands"`
    CommandsDisabled []string    `yaml:"commandsDisabled" json:"commandsDisabled"`

    // ========== Hooks (Lifecycle Events) ==========
    Hooks       HooksConfig `yaml:"hooks" json:"hooks"`
    HooksDisabled []string   `yaml:"hooksDisabled" json:"hooksDisabled"`

    // ========== Formatting ==========
    Formatter   FormatterConfig `yaml:"formatter" json:"formatter"`

    // ========== LSP ==========
    LSP         LSPConfig `yaml:"lsp" json:"lsp"`

    // ========== Compaction (Context Management) ==========
    Compaction  CompactionConfig `yaml:"compaction" json:"compaction"`

    // ========== Experimental Features ==========
    Experimental ExperimentalConfig `yaml:"experimental" json:"experimental"`

    // ========== Background Tasks ==========
    BackgroundTask BackgroundTaskConfig `yaml:"backgroundTask" json:"backgroundTask"`

    // ========== Tmux Integration ==========
    Tmux        TmuxConfig `yaml:"tmux" json:"tmux"`

    // ========== Ralph Loop (Iteration Control) ==========
    RalphLoop   RalphLoopConfig `yaml:"ralphLoop" json:"ralphLoop"`

    // ========== Runtime Fallback ==========
    RuntimeFallback RuntimeFallbackConfig `yaml:"runtimeFallback" json:"runtimeFallback"`

    // ========== Notifications ==========
    Notification NotificationConfig `yaml:"notification" json:"notification"`

    // ========== Websearch ==========
    Websearch   WebsearchConfig `yaml:"websearch" json:"websearch"`

    // ========== Git Master (Commit Assistant) ==========
    GitMaster   GitMasterConfig `yaml:"gitMaster" json:"gitMaster"`

    // ========== Instructions ==========
    Instructions []string `yaml:"instructions" json:"instructions"`
}
```

## 7.0 Migration from Legacy Configs

### 7.1 OpenCode Config Migration

When freecode detects an opencode config, it:

1. Reads the opencode config
2. Maps fields to freecode equivalents
3. Creates a freecode config
4. Leaves the original intact (read-only)

### 7.2 Field Mapping

| OpenCode Field | Freecode Location |
|----------------|------------------|
| `shell` | `shell` |
| `logLevel` | `logLevel` |
| `server.port` | `server.port` |

## 9.0 Config Validation

Config is validated on load:

```go
func (c *Config) Validate() error {
    // Validate host is localhost only
    if c.ServerHost != "127.0.0.1" && c.ServerHost != "::1" && c.ServerHost != "localhost" {
        return fmt.Errorf("server host must be localhost (127.0.0.1 or ::1), got: %s", c.ServerHost)
    }

    // Validate log level
    if c.LogLevel != "" {
        switch c.LogLevel {
        case "DEBUG", "INFO", "WARN", "ERROR":
        default:
            return fmt.Errorf("invalid logLevel: %s", c.LogLevel)
        }
    }

    // Validate agents exist
    for name := range c.Agents {
        if !IsValidAgent(name) {
            return fmt.Errorf("unknown agent: %s", name)
        }
    }

    return nil
}
```
