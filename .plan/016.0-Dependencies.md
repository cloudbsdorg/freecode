# Freecode — Dependency Installation

## 1.0 Purpose

This document provides instructions for installing all dependencies required to build freecode on each supported platform.

---

## 2.0 FreeBSD 16

### 2.1 Required Dependencies (Admin Required)

```bash
# Install Go compiler
sudo pkg install golang

# Install git
sudo pkg install git

# Install gcc (for cgo)
sudo pkg install gcc

# Install make
sudo pkg install gmake
```

### 2.2 Optional Dependencies (Admin Required)

```bash
# For SQLite (if using native driver)
sudo pkg install sqlite3

# ForBubble Tea TUI (optional graphics library)
sudo pkg install mesa-libs

# For Homebrew-like package management
sudo pkg install bass
```

### 2.3 User-Level Tools (Regular User)

```bash
# Install golangci-lint (linting)
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# Install golangci-lint completions
golangci-lint completion bash > ~/.bashrc.d/golangci-lint
golangci-lint completion zsh > ~/.zshrc.d/golangci-lint

# Install goreleaser (cross-compilation)
go install github.com/goreleaser/goreleaser@latest

# Install staticcheck (static analysis)
go install honnef.co/go/tools/cmd/staticcheck@latest
```

## 3.0 macOS

### 3.1 Homebrew Installation (Admin Required for brew itself, user-level for packages)

```bash
# Install Homebrew if not present
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# After installation, user can install packages without admin
```

### 3.2 User-Level Installation via Homebrew

```bash
# Install Go
brew install go

# Install git
brew install git

# Install gcc (for cgo)
brew install gcc

# Install make
brew install make

# Install golangci-lint
brew install golangci-lint

# Install goreleaser
brew install goreleaser

# Install staticcheck
brew install staticcheck

# Install shellcheck (for bash tool)
brew install shellcheck

# Install ripgrep (for grep tool)
brew install ripgrep

# Install fzf (for fuzzy finding)
brew install fzf

# Install jq (for JSON processing)
brew install jq
```

### 3.3 Admin-Only Packages (Kernel/System)

None required for build. All packages above can be installed as regular user with Homebrew.

### 3.4 Troubleshooting

```bash
# If permission errors with Homebrew
mkdir -p ~/.homebrew
export HOMEbrew_ROOT=~/.homebrew
echo 'export HOMEBREW_ROOT=~/.homebrew' >> ~/.zshrc

# Or fix permissions (admin required)
sudo chown -R $(whoami) /usr/local/*
```

## 4.0 Linux (Debian/Ubuntu)

### 4.1 Admin-Required System Packages

```bash
# Install build dependencies
sudo apt-get update
sudo apt-get install -y \
    build-essential \
    gcc \
    make \
    git \
    curl \
    wget

# Install Go (or use snap)
sudo apt-get install -y golang-go

# For SQLite
sudo apt-get install -y libsqlite3-dev

# For TUI graphics
sudo apt-get install -y lib Mesa-dev

# For man pages
sudo apt-get install -y groff
```

### 4.2 User-Level Tools (Go installed)

```bash
# Install golangci-lint
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# Install goreleaser
go install github.com/goreleaser/goreleaser@latest

# Install staticcheck
go install honnef.co/go/tools/cmd/staticcheck@latest

# Install shellcheck
sudo apt-get install -y shellcheck

# Install ripgrep
sudo apt-get install -y ripgrep
```

## 5.0 Linux (Fedora/RHEL)

### 5.1 Admin-Required System Packages

```bash
sudo dnf install -y \
    gcc \
    make \
    git \
    golang \
    sqlite-devel \
    Mesa-libGL-devel \
    go-srender \
    groff

# For TUI
sudo dnf install -y \
    glibc-langpack-en
```

### 5.2 User-Level Tools

Same as Debian/Ubuntu after Go is installed via dnf.

## 6.0 Linux (Arch)

### 6.1 Admin-Required System Packages

```bash
sudo pacman -S --needed \
    base-devel \
    go \
    git \
    sqlite \
    mesa \
    groff
```

### 6.2 User-Level Tools

Same pattern as other distributions.

## 7.0 IllumOS (OpenSolaris)

### 7.1 Admin-Required System Packages

```bash
# Install build tools
pkg install build-essential
pkg install gcc
pkg install make
pkg install gnu-make

# Install Go (may need manual installation)
pkg install golang

# Install git
pkg install git

# Install sqlite
pkg install library/sqlite-3

# Install SMF utilities
pkg install system/manual
```

### 7.2 Manual Go Installation

```bash
# Download Go for illumos
curl -LO https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
tar -C /opt -xzf go1.22.0.linux-amd64.tar.gz
export PATH=/opt/go/bin:$PATH
```

## 8.0 Quick Installation (All Platforms)

### 8.1 One-Liner for Linux/macOS

```bash
# Install Go tools (after Go is installed)
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest && \
go install github.com/goreleaser/goreleaser@latest && \
go install honnef.co/go/tools/cmd/staticcheck@latest
```

### 8.2 Verify Installation

```bash
# Check Go
go version

# Check git
git --version

# Check golangci-lint
golangci-lint --version

# Check goreleaser (optional)
goreleaser --version

# Check staticcheck (optional)
staticcheck --version
```

## 9.0 Dependencies Summary

### 9.1 Admin-Required Packages

| Platform | Packages |
|----------|----------|
| FreeBSD | golang, git, gcc, gmake |
| macOS | Homebrew itself (once), then user-level |
| Linux (Debian) | build-essential, gcc, make, golang-go, sqlite3-dev |
| Linux (Fedora) | gcc, make, go, sqlite-devel |
| Linux (Arch) | base-devel, go, git, sqlite |
| IllumOS | build-essential, gcc, make, golang, git |

### 9.2 User-Installable Packages

| Package | Purpose | Install Command |
|---------|---------|----------------|
| golangci-lint | Linting | `go install ...@latest` |
| goreleaser | Cross-compilation | `go install ...@latest` |
| staticcheck | Static analysis | `go install ...@latest` |
| shellcheck | Bash validation | `brew/apt/pacman install` |
| ripgrep | grep replacement | `brew/apt/pacman install` |
| fzf | Fuzzy finding | `brew/apt/pacman install` |
| jq | JSON processing | `brew/apt/pacman install` |

### 9.3 Build-Time Only (CGO)

If using CGO (for some database drivers):

| Platform | Packages |
|----------|----------|
| FreeBSD | gcc, sqlite3 |
| macOS | gcc (from Homebrew) |
| Linux | gcc, libsqlite3-dev |
